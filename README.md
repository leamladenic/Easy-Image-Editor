# Easy-Image-Editor
Image Editor in Python.

- Grayscale

- Histogram and cumulative histogram

- Contrast stretching

- Histogram equalization

- Gamma correction

- Amplitude segmentation

- Convolution

- Corelation

- Median filter

- Image sharpening

- Edge detection

- Image combining

- Morphological Operations (erosion, dilation, opening, closing)



This code is for Linux.
